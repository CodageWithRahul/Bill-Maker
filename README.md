🚀 Bill Maker v1.3.0 – Latest Release
Version: v1.3.0
Release Date: 27/02/2025

🔧 Tailored for Internet Service Providers
Bill Maker is an invoicing application purpose-built for Internet Service Providers (ISPs), helping manage customer billing for both broadband and cable services with ease. Unlike general billing tools, it focuses on the unique needs of ISPs.

🆕 What’s New in v1.3.0
📺 Cable Customer Billing Support – In addition to broadband, ISPs can now generate invoices for cable services.

✏️ Edit Existing Data – Easily update and correct customer details.

📊 Export Bills to Excel – Save invoices as Excel files for organized record-keeping.

💧 Watermark Invoices – Add a professional watermark to your printed bills.

🧾 More Billing Fields – Additional fields included for detailed, customized invoices.

🌟 Key Features
Invoice Generator – Create and save polished, professional invoices.

Company Settings – Enter and update your business details (name, address, contact).

Custom Terms & Conditions – Add up to 4 custom rules to every invoice.

Simple Interface – Clean, intuitive layout designed for efficiency and clarity.

⚙️ Improvements in v1.3.0
⚡ Better Performance – Handles larger records more smoothly.

📦 Reduced EXE Size – Optimized for faster download and better system compatibility.

🖨️ Improved Print Layout – Invoice print formatting refined for clarity and alignment.
